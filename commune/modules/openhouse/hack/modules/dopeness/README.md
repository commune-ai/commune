## Usage

```python

# Create a new Dopeness instance with custom vibe level
dope = Dopeness(vibe_level=8, chill_factor=0.7)

# Generate a random vibe phrase

# Perform a vibe check

print(result)

# Enhance text with dope vibes

print(enhanced)

# Get a vibe playlist

dope.chill_out(0.5)
```
